vue框架的深入了解

vue框架为我们开发带来了很多的方便之处，很够系统的明白vue的原理还是很有必要的。通过对于vue，vuex，vue-router源码分析提高对于vue的认识；

vue响应式系统，实现MVVM模型的框架

源码主要用es语法加载着 flow语法

  • src/compiler: 主要做模板解析渲染的
  • core 肯定就是核心代码了
    • instance
    • observer
    • util
    • vdom

vuex

vuex通过install按照第三方插件的方式引入的。
对于低于2.0的版本做一个处理,高版本直接通过vue挂载，低的直接作为vue的属性挂载

store.js定义了vuex的核心代码组成

vue-router

也是按照第三方插件的方式引入的。

history 改变触发的事件：

  • 对于 hash 模式而言触发 window 的 hashchange 事件，
  • 对于 history 模式而言则触发 window 的 popstate 事件。
