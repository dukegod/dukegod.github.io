git 进阶

git-config 相关的说明

通过修改此作为我们登陆的凭证，分为全局与本地

修改提交用户名或者邮箱

全局的通过vim ~/.gitconfig来查看，修改全局可以为每一次的提交省去登陆，提交人的相关信息

局部的通过当前路径下的 .git/config文件来查看
对于不用的项目，又不想修改全局的属性，可以修改本地的属性，实现设置登陆与提交信息的相关的操作。

也可以修改提交的用户名和Email
