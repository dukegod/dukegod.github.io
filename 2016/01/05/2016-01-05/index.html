支持disqus

第一步肯定是注册一个账号

然后点击 setting > Add Disqus To Site 进入另一个页面 > 点击用户名旁边有个[ install on your site] > site name: 输入你的网站; 你就可以看到所谓的’short_name’

如果不行的话，进入 setup 页面 找到 website name 与 website url 重新输入，然后保存下

然后修改_config.yml
