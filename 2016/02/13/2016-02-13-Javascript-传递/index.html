传递

按值传递

对基本类型，这个值是原始值

基本数据类型：Null/Undefined/String/Number/Boolean

引用传递

javascript中的参数传递都采用 按值传递的方式
对于对象来说，这个值是指对象的内存地址

object

array

都是按照值传递，对于引用传递，地址改变了，所有的指向此内存的都改变了。
