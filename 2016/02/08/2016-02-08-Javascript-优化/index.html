优化

关于web端的性能优化，大致分为：

  • 页面优化
  • 代码优化
  • 服务器优化

页面优化

  • 减少HTTP请求
  • 使用CDN
  • 图片使用webp，最好标注width，height，避免重绘

代码优化

  • 合并 JavaScript、CSS 等文件
  • 压缩代码

服务优化

  • nginx配置Gzip

  • 减少 DNS 查询
    用户输入 URL 以后，浏览器首先要查询域名（hostname）对应服务器的 IP 地址，一般需要耗费 20-120 毫秒 时间。DNS 查询完成之前，浏览器无法从服务器下载任何数据。
