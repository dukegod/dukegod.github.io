mac 下一些快捷的操作习惯

快捷键

  • CONTRLO + SPACE : SPOTLIGHT
  • C + R : 浏览器刷新
  • C+O+J(I): 打开审查元素
  • option + esc : 开始朗读信息
  • fn + fn : 开始听写
  • CMD + i ： 修改权限
  • cmd + up/down 返回上一级和进入文件
  • Command +Option+V ： 实现剪切
  • Command + shift + . : 打开隐藏文件
