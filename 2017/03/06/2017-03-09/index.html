也许这真的是奇迹，但中间的苦楚又只有多少人能够体会。

恩叔，下个赛季可能就不会指教巴萨了，这个就是指教豪门的压力，当我们看到惊天逆转的时候，我们可能想到了恩叔，看到了恩叔的激情庆祝，但是恩叔是不是还会继续指教我萨不知道的事情。

最后我们看到6-1的比分，有多少人是满眼泪水，为这种全心投入，为这种对于足球的爱的坚定！
