书写基于ES6的npm包问题

之前写的npm的发布包都不是基于ES6的，没有遇到在与其他的包一起使用出现，对于ES6的相关问题。
今天用es6写了一个包，webpack引入的使用总是报错。

就是因为没有处理ES6的原因。

通过webpack的配置的解决方案

  • 引入babel-loader: 不用说了吧
  • 打包为UMD方式: 配置在output选项中
