盒模型:

+ border
+ padding
+ margin

box-sizing: border-box, content-box

content-box(默认): 元素的宽度/高度（width/height）等于元素边框宽度（border）加上元素内边距（padding）加上元素内容宽度/高度（content width/height）;

border-box : 内容宽度/高度=width/height-border-padding
